# Clarifications — Hệ thống giải thưởng

MoMorph screen: https://momorph.ai/files/9ypp4enmFmdK3YAFJLIu6C/screens/zFYDgyj_pD

## Session 2026-07-16

- Q: The spec calls the route `/awards` and public — is that as-shipped? → A: No. It ships at **`/award-information`** and is **auth-gated**, not public. Both the header "Award Information" tab and the "standards" nav entry resolve here (`NAV_HREFS.awards` and `NAV_HREFS.standards` in `lib/saa/content.ts` both point to `/award-information`). The page runs the same gate as the homepage — `supabase.auth.getUser()` + the `DEMO_COOKIE` stand-in, `if (!user && !hasDemoSession) redirect("/login")` — and is covered by `proxy.ts`. The spec's `PUBLIC_ROUTES` / `src/middleware.ts` note is stale (that was a different stack).
- Q: The spec lists localization as out of scope (VN content only) — did that hold? → A: No. The page is **fully VN/EN** via `saa_lang` + `getDict()`. `lib/saa/awards.ts` carries **structural data only** (slug, proper-name titles, medal art, quantity numbers, prize amounts, `medalSide`); all prose — paragraphs, quantity `unit`, prize `notes` — lives in `dict.awardInfo.awards`, keyed by the same slug and **index-aligned to `prizes`**. Section labels ("Số lượng giải thưởng:", "Giá trị giải thưởng:", "Hoặc") are dict keys too.
- Q: The spec wants a dropdown selector replacing the sidebar on mobile — is there one? → A: No separate SP dropdown. A **single responsive `AwardNav` rail** is used at every breakpoint (sticky `lg:sticky lg:top-24 lg:w-56`, stacking above the card list on mobile). It scroll-spies via `IntersectionObserver` (`rootMargin: "-30% 0px -60% 0px"`) and smooth-scrolls through `href="#{slug}"` anchors + CSS `scroll-behavior`. The dedicated mobile dropdown selector is deferred.
- Q: What renders the hero keyvisual? → A: A **documented CSS placeholder** (`KeyvisualHero`) — layered radial-gradient colour ribbons plus the `ROOT FURTHER` wordmark (`/saa/root-further.png`). The original artwork ("image 20") is a flat raster fill, not an `MM_MEDIA` node, so it isn't downloadable, and the Figma render API was down at build time. Swap in the real export when available.
- Q: How is the Signature 2025 – Creator dual prize handled? → A: `prizes` is an array; Signature has **two entries** (`5.000.000 VNĐ`, `8.000.000 VNĐ`). `AwardCard` renders a "Hoặc" separator before entries after the first, and `dict.awardInfo.awards["signature-2025-creator"].notes` supplies the two index-aligned notes ("cho giải cá nhân", "cho giải tập thể").
- Q: Where do the six award medals come from? → A: One shared orb backdrop (`/saa/award-bg.png`) with a per-award gold name PNG overlaid inside it (`/saa/award-<slug>.png`, curated set reused from the homepage). `AwardMedal` composes the two `next/image` layers; there is no distinct 336×336 image per award.
